## Project 01

In the `project_01/` folder you can find an exploration in trends for U.S. births 
between 2000 and 2014 using daily birth count data. The goal is to tell a story about how 
birth patterns vary across different time scales (yearly, monthly, and by day of the week) and what events might explain those patterns. The dataset contains one row per day
from January 1, 2000 through December 31, 2014. Each row includes the year, month, date of month, full date, day of week, and number of births recorded that day in the United States.
